# Шпаргалка по Git


## Команды Git на примере `GitCheatSheet`
### Создание репозитория
1. Создать каталог для проекта
1. Перейти в него
1. Инициализировать репозиторий

```console
$ mkdir GitCheatSheet
$ cd GitCheatSheet/
$ git init
```

## Удаление репозитория
1. Перейти в каталог с репозиторием
1. Удалить каталог `.git`

```console
$ cd GitCheatSheet
$ rm -rf .git
```
